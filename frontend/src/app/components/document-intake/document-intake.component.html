Ingestion Guidelines

The Digital File Library maintains rigorous security and document visibility rules:

Security Protection Active:

For security, the extraction pipeline is protected. Every ingest request is audited, parsed, and restricted to staff roles by default. Public citizens see only documents that are explicitly flagged as isPublished: true.

Automatic Cache Sync:

Once Docling extracts layout and chunks are saved, a background webhook triggers automatically to cache metadata to `eagle-api`, keeping legacy platforms synced.

AI Text Extractor:

Docling isolates headers, paragraphs, and tables automatically, indexing individual pages as independent chunks for high-relevance deep keyword search.
